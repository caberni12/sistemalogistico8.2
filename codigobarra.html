⚙️ Configuración del generador

📐 Medida, papel y formato

La etiqueta se arma centrada dentro del ancho y alto configurado. El sistema reduce texto, QR o barras si el espacio es pequeño para evitar que se salga.

🔗 Conexión Apps Script / MAESTRA

La sincronización consulta Apps Script y guarda la MAESTRA en IndexedDB. La acción POST se usa para enviar productos pegados o la MAESTRA local hacia la BD remota.
La administración de MAESTRA permite ver, buscar, agregar, editar y eliminar códigos guardados en memoria local.
Para rollos de 5 cm de ancho x 3 cm de alto: el sistema usa ANCHO 50 mm y ALTO 30 mm, arma dos etiquetas por fila, aplica separación central entre columnas y separación vertical hacia abajo por cada etiqueta.
Modo intervalo deja separación externa real entre etiquetas, sin crear saltos de página. Para etiquetas pequeñas 30 x 50 mm puedes activar doble intervalo: mantiene la etiqueta sin margen interno y duplica solo la separación externa.
Al imprimir, el sistema respeta el ancho y alto real de la etiqueta en milímetros y escala el contenido para ocupar mejor el espacio interno sin salirse.

☁️ Enviar pegado a MAESTRA remota

Valida el bloque pegado y envíalo por POST al Apps Script configurado. Modo seguro: no borra la MAESTRA de destino; agrega códigos nuevos y reemplaza solo los códigos que ya existen.

Pendiente: pega datos, valida y presiona “Enviar pegado a MAESTRA remota”. La operación no elimina registros existentes.
Regla aplicada: CODIGO existente = se reemplaza esa fila; CODIGO nuevo = se agrega al final; CODIGO remoto no enviado = queda intacto.
