📐 Medida, papel y formato

La etiqueta se arma centrada dentro del ancho y alto configurado. El sistema reduce texto, QR o barras si el espacio es pequeño para evitar que se salga.

🔗 Conexión Apps Script / MAESTRA

La sincronización consulta Apps Script y guarda la MAESTRA en IndexedDB, una memoria local grande. Así evita el error de cuota de localStorage y permite buscar código y descripción rápidamente sin consultar la URL en cada búsqueda.
Modo intervalo deja separación externa real entre etiquetas, sin crear saltos de página. Para etiquetas pequeñas 30 x 50 mm puedes activar doble intervalo: mantiene la etiqueta sin margen interno y duplica solo la separación externa.
Al imprimir, el sistema respeta el ancho y alto real de la etiqueta en milímetros y escala el contenido para ocupar mejor el espacio interno sin salirse.

📂 Opciones del importador masivo
