CAMBIOS APLICADOS - 2026-05-04

1) Apps Script
- Se agregó APPS_SCRIPT_MOVIMIENTO_ALERTAS_LISTO.gs.
- Debes copiar ese archivo completo en tu proyecto de Google Apps Script y publicar una nueva implementación del Web App.
- El Apps Script acepta ahora las acciones:
  agregar / guardar_movimiento / agregar_movimiento / guardar_ubicacion
  editar / editar_movimiento / actualizar_movimiento / actualizar_ubicacion
  eliminar / eliminar_movimiento / borrar_movimiento / eliminar_ubicacion
  listar / listar_movimiento / listar_ubicaciones / listar_bd&sheet=MOVIMIENTO
- La hoja usada para el módulo Ubicaciones es MOVIMIENTO.
- Las alertas se escriben con token único en PEDIDOS y registro en ALERTAS_PEDIDOS.
